<action>
Update SQL file execution in cmd/exec.go:

1. Check --autocommit flag:

2. If autocommit=true (default):
   - Each statement runs in its own implicit transaction
   - No explicit BEGIN/COMMIT needed
   - GORM default behavior

3. If autocommit=false:
   - Start explicit transaction: tx := db.Begin()
   - Execute all statements within tx
   - On success: tx.Commit()
   - On error: tx.Rollback()
   - Check for BEGIN/COMMIT/ROLLBACK in SQL:
     - If present: passthrough to database (nested transaction or error)

4. Update flag:
   - execCmd.Flags().Bool("autocommit", true, "Auto-commit each statement")
</action>
