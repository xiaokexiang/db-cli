<name>Task 1: Create schema inspection helpers</name>
<files>internal/database/schema.go</files>
<action>
Create internal/database/schema.go with MySQL schema inspection functions:

1. GetTableColumns(db *gorm.DB, tableName string) ([]map[string]interface{}, error)
   - Executes: SELECT COLUMN_NAME, DATA_TYPE, CHARACTER_MAXIMUM_LENGTH, IS_NULLABLE, COLUMN_DEFAULT, EXTRA
             FROM INFORMATION_SCHEMA.COLUMNS
             WHERE TABLE_SCHEMA = ? AND TABLE_NAME = ?
   - Returns columns: Field, Type, Null, Key, Default, Extra (matching MySQL DESCRIBE format)

2. GetIndexes(db *gorm.DB, tableName string) ([]map[string]interface{}, error)
   - Executes: SHOW INDEX FROM tableName
   - Returns: Column_name, Index_name, Non_unique, Seq_in_index

3. GetForeignKeys(db *gorm.DB, tableName string) ([]map[string]interface{}, error)
   - Executes: SELECT COLUMN_NAME, CONSTRAINT_NAME, REFERENCED_TABLE_NAME, REFERENCED_COLUMN_NAME
             FROM INFORMATION_SCHEMA.KEY_COLUMN_USAGE
             WHERE TABLE_SCHEMA = ? AND TABLE_NAME = ? AND REFERENCED_TABLE_NAME IS NOT NULL
   - Returns: Column, Constraint, ReferencedTable, ReferencedColumn

4. ListDatabases(db *gorm.DB) ([]map[string]interface{}, error)
   - Executes: SHOW DATABASES
   - Returns: Database

5. ListTables(db *gorm.DB) ([]map[string]interface{}, error)
   - Executes: SHOW TABLES
   - Returns: Tables_in_<database>

All functions use GORM's db.Raw() and ScanRows for consistent result handling.

<action>
Create cmd/desc.go with descCmd implementing:

1. Flags:
   - --table (string): Table name to describe
   - --indexes (bool): Show indexes for the table
   - --foreign-keys (bool): Show foreign keys for the table
   - --databases (bool): List all databases
   - --tables (bool): List all tables in current database

2. Command logic in runDesc function:
   - Validate required connection parameters (user, database for --table/--indexes/--foreign-keys/--tables)
   - Open database connection using database.OpenConnection(cfg)
   - defer database.CloseConnection(db)
   - Route based on flags:
     - --databases: call database.ListDatabases, output as JSON
     - --tables: call database.ListTables, output as JSON
     - --table=xxx: call database.GetTableColumns, output as JSON
     - --table=xxx --indexes: call database.GetIndexes, output as JSON
     - --table=xxx --foreign-keys: call database.GetForeignKeys, output as JSON

3. Error handling:
   - Return error if no flags provided
   - Return error if conflicting flags (e.g., --databases and --table together)
   - Use same error pattern as exec.go (fmt.Fprintf to stderr, non-zero exit)

4. Dameng support (CONN-03 preparation):
   - In schema.go functions, check cfg.DBType
   - If "dameng", return error: "dameng schema inspection not yet supported in Phase 2"
   - Keep infrastructure ready for Phase 4 implementation
