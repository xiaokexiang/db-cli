## v1 Requirements - All Complete

### Connection (CONN)

- [x] **CONN-01**: 支持通过 flag 指定数据库连接参数 — **v1.0** ✅
  - Flags: `-h/--host`, `-P/--port`, `-u/--user`, `-p/--password`, `-d/--database`, `-t/--type`
  - 数据库类型：`mysql`, `dameng`
  - **Outcome:** Validated — Phase 1 delivery, later simplified to -c DSN URL only in Phase 5

- [x] **CONN-02**: MySQL 数据库连接 — **v1.0** ✅
  - 使用 GORM + go-sql-driver/mysql v1.9.3
  - 支持 MySQL 5.7+ 和 8.x
  - **Outcome:** Validated — Core functionality, all tests pass

- [x] **CONN-03**: 达梦数据库连接 — **v1.0** ✅
  - 使用 GORM + github.com/godoes/gorm-dameng v0.7.2（纯 Go 实现，无需 CGO）
  - 支持达梦 DM8+
  - DSN 格式：dm://user:password@host:port?schema=database
  - 默认端口：5236
  - **Outcome:** Validated — Phase 4 delivery, pure Go driver works without CGO

### Execution (EXEC)

- [x] **EXEC-01**: 执行单条 SQL 语句 — **v1.0** ✅
  - 命令：`db-cli exec -c 'mysql://...' 'SELECT ...'`
  - 支持所有 SQL 类型（SELECT/INSERT/UPDATE/DELETE/DDL）
  - **Outcome:** Validated — Core functionality

- [x] **EXEC-02**: 执行 SQL 文件 — **v1.0** ✅
  - 命令：`db-cli exec -c 'mysql://...' --file=xxx.sql`
  - 支持大文件（逐条解析执行）
  - **Outcome:** Validated — Includes line number tracking for errors

- [x] **EXEC-03**: 错误处理 — **v1.0** ✅
  - SQL 文件执行遇到错误立即停止
  - 显示错误码和错误信息（含行号）
  - 返回非零退出码
  - **Outcome:** Validated — ExecutionError struct with line tracking

- [x] **EXEC-04**: 事务控制 — **v1.0** ✅
  - `--autocommit=true`（默认）：每条 SQL 自动提交
  - `--autocommit=false`：SQL 文件内的所有语句在一个事务中执行
  - **Outcome:** Validated — Fixed bug: transaction wraps ALL statements, not per-statement

### Query (DQL)

- [x] **DQL-01**: JSON 格式输出 — **v1.0** ✅
  - 默认输出格式
  - 数组包裹，每行一个 JSON 对象
  - **Outcome:** Validated — Default output format

- [x] **DQL-02**: 多格式支持 — **v1.0** ✅
  - `--format=json`（默认）
  - `--format=table`：ASCII 表格
  - `--format=csv`：CSV 格式
  - **Outcome:** Validated — Phase 2 delivery, pure Go implementation

### Schema Inspection (DESC)

- [x] **DESC-01**: 查看表结构 — **v1.0** ✅
  - 命令：`db-cli desc -c '...' --table=xxx`
  - 显示字段名、类型、长度、是否为空、默认值
  - **Outcome:** Validated

- [x] **DESC-02**: 查看索引 — **v1.0** ✅
  - `--indexes` flag：显示表的索引信息
  - **Outcome:** Validated

- [x] **DESC-03**: 查看外键 — **v1.0** ✅
  - `--foreign-keys` flag：显示外键约束
  - **Outcome:** Validated

- [x] **DESC-04**: 查看元数据 — **v1.0** ✅
  - `--databases`：列出所有数据库
  - `--tables`：列出所有表
  - **Outcome:** Validated

### Import/Export (IO)

- [x] **IO-01**: 导入 SQL 文件 — **v1.0** ✅
  - 命令：`db-cli import -c '...' --file=xxx.sql`
  - 等价于 `exec --file`，语义更清晰
  - **Outcome:** Validated — Thin wrapper around exec --file

- [x] **IO-02**: 导出查询结果 — **v1.0** ✅
  - 命令：`db-cli export -c '...' --query='SELECT ...' --output=file.sql`
  - 支持 `--format=insert`（生成 INSERT 语句）
  - 支持 `--format=ddl`（生成建表语句）
  - **Outcome:** Validated

- [x] **IO-03**: 导出整表 — **v1.0** ✅
  - 命令：`db-cli export -c '...' --table=xxx --output=file.sql`
  - 导出表结构和数据
  - **Outcome:** Validated

### Logging (LOG)

- [x] **LOG-01**: 命令历史 — **v1.0** ✅
  - 记录执行过的命令（不含密码）
  - 密码字段显示为 `***`
  - 存储位置：`~/.db-cli/history.log`
  - **Outcome:** Validated — JSON lines format, log rotation at 10MB

- [x] **LOG-02**: 错误日志 — **v1.0** ✅
  - 记录错误信息到 `~/.db-cli/error.log`
  - 包含时间戳、命令、错误码、错误信息
  - **Outcome:** Validated

### Skill Integration (SKILL)

- [x] **SKILL-01**: Claude Code Skill — **v1.0** ✅
  - 解析自然语言为 db-cli 命令
  - 5 个工具：count, desc, export, import, exec
  - 模板匹配系统（D-09, D-10）
  - **Outcome:** Validated — Template matching over LLM generation (safer, more controllable)

- [x] **SKILL-02**: 自动下载 — **v1.0** ✅
  - Skill 安装时从 GitHub Releases 下载 db-cli 二进制
  - 支持 6 个平台：windows/amd64, windows/arm64, darwin/amd64, darwin/arm64, linux/amd64, linux/arm64
  - `npx db-cli-skill install` 一键安装
  - **Outcome:** Validated — CI/CD workflow for automated builds

## Out of Scope (v1.0 Decisions)

- **交互式 REPL 模式** — 用户偏好纯子命令设计，CI/CD 友好
- **环境配置管理** — 每次执行时手动指定连接参数，安全考虑
- **密码存储** — 不保存敏感信息到配置文件
- **AI 内置** — CLI 本身不提供 AI 能力，由 Skill 层负责
- **安全限制** — 不拦截 CRUD 操作，信任配置即权限
- **连接池配置** — 使用 GORM 默认设置
