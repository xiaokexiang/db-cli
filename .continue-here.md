## 测试验证结果

**MySQL 测试通过:**
```bash
# exec - table/json/sql 格式
node dist/index.js exec -c 'mysql://root:123456@10.50.8.44:3306/mysql' 'SELECT 1 AS test'
node dist/index.js exec -c 'mysql://...' --format=json 'SELECT 1 AS num'
node dist/index.js exec -c 'mysql://...' --format=sql 'SELECT 1 AS num'

# desc - databases/tables/table structure
node dist/index.js desc -c 'mysql://...' --databases
node dist/index.js desc -c 'mysql://...' --tables
```

**Dameng 测试通过:**
```bash
# exec
node dist/index.js exec -c 'dameng://SYSDBA:SYSDBA001@10.50.8.44:5236' 'SELECT 1 AS test FROM DUAL'

# desc - databases/tables
node dist/index.js desc -c 'dameng://...' --databases  # 返回 SYS, SYSAUDITOR, SYSDBA, SYSSSO
node dist/index.js desc -c 'dameng://...' --tables     # 返回表列表
node dist/index.js desc -c 'dameng://...' -t ACCESSKEY # 返回表结构
```

## 关键技术决策

1. **使用 dmdb 官方驱动** - 达梦数据库官方 Node.js 驱动 (npm install dmdb)
2. **DSN URL 格式统一** - `type://user:pass@host:port/database`
3. **输出格式统一** - table/json/sql 三种格式
4. **无配置文件** - 安全考虑，每次手动指定连接参数

## 注意事项

1. **达梦数据库加密** - loginEncrypt 默认为 false，如需加密需服务器端配置
2. **大小写敏感** - 达梦表名默认大写
3. **DUAL 表** - 达梦查询需要使用 FROM DUAL
4. **驱动依赖** - dmdb 是达梦官方驱动，需要确保版本兼容
